# Echozy — Modern Social Media Platform

Echozy is a full stack social media platform built with the MERN stack. Share posts, connect with friends, send direct messages, discover communities and chat with Echo AI powered by Gemini.

## 🌐 Live Demo
- **Website:** https://echozy.vercel.app
- **About Echozy:** https://echozy.vercel.app/about-prerender

## ✨ Features
- 📝 Share posts and photos
- 👥 Friend connections and requests
- 💬 Direct messaging between friends
- 🤖 Echo AI assistant powered by Gemini
- 👤 Customizable user profiles
- 🔔 Real-time notifications

## 🛠️ Built With
- **Frontend:** React, Tailwind CSS, Framer Motion
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **AI:** Google Gemini API
- **Deployment:** Vercel
